Whydah

Whydah is a light weight user administration and single sign-on component developed in Oslo, Norway and supported by Cantara.

It is subdivided in five logical components where four are built using Java. Documentation is found at wiki.cantara.no/display/iam/whydah

This is the fifth component, a simple user- and application administration tool to make it easy to browse and modify users.

This component is built with:

  • Angular.js
  • Bootstrap / Bootswatch Yeti
  • jQuery
  • Components from UI Bootstrap
